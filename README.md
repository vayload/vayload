Vayload is a next-generation, headless CMS built for developers who value performance and clarity.
It harnesses the speed of Go on the backend, the flexibility of Lua for scripting and extensibility, and the fluid reactivity of Svelte for a modern, streamlined admin experience.

Designed to be fast, modular, and fully scriptable, Vayload gives you complete control over your content, APIs, and workflows—without sacrificing performance or developer experience.
